Introduction

Hello, the name's Sherly. This is my personal HTML website that I mostly coded by myself. The purpose of this website is for it to act like a journal for me. If you found this, it means I either gave you permission or you're snooping. There's no inbetween.

This HTML website officialy started on 5th of March, 2023. Working on this website will become my new hobby that I hope to keep.
